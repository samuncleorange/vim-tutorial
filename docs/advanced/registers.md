# 寄存器与宏

这是 Vim 真正能够让你“起飞”的功能。

## 寄存器 (Registers)
Vim 有多个剪贴板，称为寄存器。
- `"{reg}`：指定寄存器。
- `"ayiw`：将当前单词复制到寄存器 `a` 中。
- `"ap`：粘贴寄存器 `a` 中的内容。
- `"+y`：复制到系统剪贴板（需要特定支持）。

## 宏 (Macros)
宏可以录制一系列按键操作并重复播放。
- `q{letter}`：开始录制到寄存器 `{letter}`。
- 执行你的操作...
- `q`：停止录制。
- `@{letter}`：播放录制的操作。
- `@@`：重复上次运行的宏。

### 批量处理示例
想象你需要给 100 行代码末尾加分号。录制一次 `A;j`，然后输入 `99@a` 即可全自动完成。
