# 为什么选择 Vim

Vim 是由 Bram Moolenaar 在 1991 年发布的文本编辑器，它是从 vi 演变而来的。

## 速度与效率
Vim 的核心哲学是：**你的手指永远不应该离开主键盘区（Home Row）**。通过模式化编辑，你可以用极短的按键组合完成复杂的修改。

## 无处不在
无论是服务器远程连接（SSH），还是在现代 IDE（VS Code, JetBrains）中，你都能找到 Vim 的身影或插件。

## 高度可定制
通过 Vimscript 或 Lua（Neovim），你可以将 Vim 打造得完全符合你的个人习惯。

> "Once you learn Vim, you'll never settle for a cursor again."
